discodb2 — badge color palette preview

Hue-only palette generated by the largest-gap bisection rule (the hue angle is the only variable; saturation & lightness are fixed). Colors arrive in a fixed, deterministic order. Set Start hue = 0 to reproduce the spec example 0, 180, 90, 270, 45, 135, 225, 315…. The default start is an artistic pick. In the real app the same badge identity shares one color, so the number of distinct colors equals the number of badge types (usually far fewer than 50).
