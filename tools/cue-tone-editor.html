Cue tone editor — discodb2 Wizard

Two presets: "during" (long low tone = reaction window) and "after" (short cue = start signal). Tune each, listen, export both.

Device sounds — connect / disconnect

Two quick notes, like the Windows USB plug/unplug chirp: connect rises low→high, disconnect falls high→low; in both the second note is slightly longer. One set of params drives both (used by the cockpit's playStartBeep / playStopBeep).

Frame-melody sonification — per step type

The live CAN-frame "melody" is shaped per PHASE so the operator hears WHICH phase they're in. Pick a voice, tune its synth + filter + effect, ▶ Play to audition. "Awaiting" is the on-input overlay (a step waiting for confirmation). All effects are native Web-Audio nodes — cheap (one filter + one LFO).

Tip: give each phase a distinct fingerprint — e.g. Noise = band-pass + fast, Stimulus = high-pass square + vibrato, Observe = low-pass sine + slow tremolo, Awaiting = heavy low-pass + tremolo. Then the operator never has to look.
